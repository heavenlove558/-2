# 抽油杆三维力学模型

基于4篇核心论文的水平井/定向井抽油杆柱三维力学分析与偏磨预测模型。

## 已实现功能

- [x] **模块1** - 三次样条井眼轨迹计算（井斜角/方位角/狗腿度/曲率半径）
- [x] **模块2** - 上下冲程9种力分量分析（重力/浮力/惯性力/液柱载荷/摩擦力/液体阻力）
- [x] **模块3** - 双平面支持力分解（狗腿平面+垂直平面）+ 摩擦力计算
- [x] **模块4** - 中和点定位 + 弯曲应力 + 偏磨风险评定
- [ ] **模块5** - 有限元扶正器布点优化（待实现）

## 运行

```bash
python main.py
```

输出图表保存在 `output/` 目录。
